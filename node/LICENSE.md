### Perlin Noise Implementation

The Perlin Noise algorithm implemented in `noise.cpp` is based on Ken Perlin's
reference implementation. The original algorithm is described in:

"Improving Noise" by Ken Perlin, SIGGRAPH 2002
https://mrl.cs.nyu.edu/~perlin/noise/

### OpenSimplex2 Implementation

The OpenSimplex2 noise functions are based on the algorithm by Kurt Spencer.
Original implementation available at:
https://github.com/KdotJPG/OpenSimplex2
