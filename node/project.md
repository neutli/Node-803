# Node Editor Project

## プロジェクト概要

このアプリケーションは、Blenderのシェーダーノードに触発されたビジュアルノードベースのテクスチャ生成ツールです。

## 目的

**生活感あふれる都市をプロシージャルに作成するため**

このプロジェクトの最終目標は、リアルで有機的な都市環境をプロシージャルに生成することです。
自然な川の流れ、地形の起伏、道路網、建物の配置など、都市を構成する要素を
ノードベースのワークフローで制御可能にすることで、手作業では困難な規模と複雑さを持つ
環境を効率的に作成できるようにします。

### 現在の機能

- **ノードベースエディタ**: ビジュアルなノード接続によるテクスチャ生成
- **ノイズテクスチャ**: Perlin、Simplex、OpenSimplex2、Ridged Multifractal など
- **Water Source ノード**: 湖や水源の生成
- **River Texture ノード**: 水源から流れる川の生成
- **カラーランプ**: グラデーションマッピング
- **マッピング・座標変換**: UV操作、スケール、回転
- **リアルタイムプレビュー**: パラメータ変更の即時反映

---

## 今後の展望

### 1. 3D機能の実装

2Dテクスチャ生成から3D地形生成への拡張を計画しています。

- **ハイトマップからメッシュ生成**: 生成したテクスチャを3D地形に変換
- **3Dビューポート**: OpenGL/Vulkanによるリアルタイム3D表示
- **地形エクスポート**: OBJ、FBX、glTF形式での出力
- **LOD (Level of Detail)**: 距離に応じたメッシュ詳細度の自動調整

### 2. ジオメトリノードシステム

Blenderのジオメトリノードに類似した機能の実装を目指します。

- **ポイント分布**: 地形上への建物、樹木、街灯などの配置
- **カーブノード**: 道路、線路、川の経路制御
- **インスタンス配置**: 効率的なオブジェクト複製
- **プリミティブ生成**: 基本形状の作成と変形
- **ブーリアン演算**: 形状の結合・減算
- **属性システム**: 頂点/面ごとのデータ管理

### 3. 都市生成機能

- **道路網生成**: L-systemsや Wave Function Collapse による道路パターン
- **区画分割**: 道路で囲まれた区画の自動認識
- **建物配置**: 区画内への建物プロシージャル配置
- **植生分布**: 公園、街路樹、庭園の自動生成
- **インフラ配置**: 電柱、標識、ベンチなどの小物配置

### 4. エクスポート・連携

- **Blender連携**: ノードグラフのBlender Geometry Nodesへの変換
- **Unreal Engine連携**: Landscapeデータとしてのエクスポート
- **Unity連携**: Terrain Dataへの出力
- **標準フォーマット**: glTF 2.0、USD対応
